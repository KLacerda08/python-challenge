Election Results

Election results were evaluated. 
Due to the large dataset, code was run first on a subset of the data (election_test.csv). 
After operability was confirmed, the csv file path was edited to reference the full data. 

The data was evalauted to provide the following :
* The total number of votes cast
* A list of candidates who received votes, with the following results:
	* The percentage of votes each candidate won
	* The total number of votes each candidate won
* The winner of the election based on popular vote.

The results were exported to file Election_Results.csv
